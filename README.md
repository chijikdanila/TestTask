# Test task

Проект выполняет все поставленные задачи тестового задания.

- Методы для выполнения второго задания находятся в файле [ExcelFilesWorker.cs](https://github.com/chijikdanila/TestTask/blob/master/TestTask/ExcelFilesWorker.cs)
- Методы для работы с базой данных находятся в файле [DatabaseWorker.cs](https://github.com/chijikdanila/TestTask/blob/master/TestTask/DatabaseWorker.cs)

Скрипт вычисления суммы и медианы [Scripts.sql](https://github.com/chijikdanila/TestTask/blob/master/TestTask/Scripts.sql)
